# Projeto de Extensão do Curso de ADS do IFSC-XXE

O projeto consiste em uma aplicação voltada a modernizar e agilizar os processos de coleta e processamento de dados de famílias atingidas por desastres naturais na regional da Defesa Civil de Xanxerê.

Site em produção: https://relatoriosoffline.app <br>
Repositório da aplicação mobile: https://github.com/Dieguito-py/relatorios-offline-mobile
